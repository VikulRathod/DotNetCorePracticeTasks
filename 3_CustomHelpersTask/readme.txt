Creating Custom Helpers

Description
You will be learning how to create and implement the Custom Tag helpers into 
your ASP.NET Core applications. 
By the end of this, you'll have the skills to the Custom Tag helper and to 
use them in the application.

Objective
Upon completion of this, you will be able to:
•	Create the Custom Tag helper in your ASP.NET Core application
•	How to use Custom tag helper in the application.


Requirements
•	Create a Login page
•	Create a custom helper for making a link highlight by passing a CSS class 
active. Make a tag helper with the name asp-active-link.
•	Create a custom tag for creating a command button with submitting behavior.
